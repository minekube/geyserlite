# Compose example

Cross-play stack: geyserlite + Paper, in 60 seconds.

```sh
docker compose up
```

Then connect:

- **Java edition**: `localhost:25565`
- **Bedrock edition**: `localhost:19132` (any Geyser-supported Bedrock version)

The Floodgate key is generated by a one-shot init container on first run.

This example uses **offline-mode** auth for simplicity. For real deployments
behind a Floodgate-aware proxy (like [Gate](https://github.com/minekube/gate)),
switch `java.auth-type` to `floodgate` and configure the proxy with the same key.

## Tearing down

```sh
docker compose down -v   # -v drops the floodgate-key + paper-data volumes
```
